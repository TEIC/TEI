Make your life easier

PSGML-Jade turns Emacs into an OpenJade frontend, providing an alternative to the command-line interface of the publishing tools. AucTeX is a package which turns Emacs into a TeX editor. Strictly speaking, AucTeX is not necessary for the SGML system proper, but with Emacs and MiKTeX you already do have 99% (disk-space-wise) of a complete TeX authoring system, so it would be a waste of functionality not to install the remaining 1%. Of course it will also help you to insert some manual pagebreaks into OpenJade's TeX backend output if this should be necessary. Finally, PSGML-DSSSL is an Emacs-Lisp file which allows you to create a skeleton DSSSL stylesheet based upon your current DTD.
