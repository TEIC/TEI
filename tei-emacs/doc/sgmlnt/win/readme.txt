SGML for Windows NT
===================

This archive contains the SGML for Windows NT tutorial in HTML format.
To read the pages, unzip all .html files into one directory and open
"book1.html" with your browser.

The latest version of this tutorial is available at:

http://ourworld.compuserve.com/homepages/hoenicka_markus/ntsgml.html

Please do not attempt to print this HTML version of the tutorial. Visit
the site above and retrieve the tutorial in one of the print formats
instead (RTF, DVI, PDF, and PostScript are available).
