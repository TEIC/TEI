att.coordinated

att.coordinated provides attributes which can be used to position their parent element within a two dimensional coordinate system.
Moduletranscr — Representation of Primary Sources
Membersline surface zone
AttributesAttributes
start start¶indicates the element within a transcription of the text containing at least the start of the writing represented by this zone or surface.
Status Optional
Datatype teidata.pointer
ulx ulx¶gives the x coordinate value for the upper left corner of a rectangular space.
Status Optional
Datatype teidata.numeric
uly uly¶gives the y coordinate value for the upper left corner of a rectangular space.
Status Optional
Datatype teidata.numeric
lrx lrx¶gives the x coordinate value for the lower right corner of a rectangular space.
Status Optional
Datatype teidata.numeric
lry lry¶gives the y coordinate value for the lower right corner of a rectangular space.
Status Optional
Datatype teidata.numeric
points points¶identifies a two dimensional area within the bounding box specified by the other attributes by means of a series of pairs of numbers, each of which gives the x,y coordinates of a point on a line enclosing the area.
Status Optional
Datatype 3–∞ occurrences of teidata.point separated by whitespace
