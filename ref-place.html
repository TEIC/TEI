<place>

<place> contains data about a geographic location [13.3.4 Places]

      model.headLike*,
      (
         model.pLike*
       | ( model.labelLike | model.placeStateLike | model.eventLike )*
      ),
      ( model.noteLike | model.biblLike | idno | linkGrp | link )*,
      ( model.placeLike | listPlace )*
   )
}
