Further Reading

The OpenJade package contains a set of .html help files in the OpenJade\jadedoc subdirectory which explain the usage and the command-line options of all tools. Henry Thompson prepared a digest of the somewhat dry DSSSL specification. Now it's also a good time to look at some introductory texts describing the concept and usage of DSSSL. A good place to start is provided by Netfolder.

Paul Prescod has compiled a very useful Introduction to DSSSL.
